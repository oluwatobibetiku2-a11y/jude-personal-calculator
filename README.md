# Jude — Personal Calculator

A small personal calculator project by BETIKU OLUWATOBILOBA JUDE.

This repo contains a single-page HTML/CSS/JavaScript calculator with two main areas:

- Arithmetic calculator (visual keypad + typing; supports +, -, ×, /, %, parentheses and decimals)
- Sets UI (build sets A & B with letters A–Z, compute union/intersection/difference)

Live demo (when published via GitHub Pages):

https://oluwatobibetiku2-a11y.github.io/jude-personal-calculator/

How to use locally

1. Open `index.html` in your browser (double-click or run `start index.html` on Windows PowerShell).
2. Use the keypad or type expressions in the arithmetic input, then press `=` to compute.
3. Click letters to build Set A / Set B and use the operation buttons.

Publishing / GitHub Pages

1. Push this branch to your GitHub repo (already set up in this workspace).
2. In GitHub → Settings → Pages, choose branch `master` (or `main`) and set folder to `/ (root)`.
3. Save — your site will be published at the URL shown in the settings (it may take a minute to build).
