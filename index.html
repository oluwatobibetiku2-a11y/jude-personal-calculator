Hello — I'm BETIKU OLUWATOBILOBA JUDE. Welcome to my personal calculator. Below is a short explanation of how the two calculator areas work so you can use them quickly.

Arithmetic: Use the arithmetic panel to type expressions or click the keypad. Supported operators are +, -, × (visual multiplication), /, parentheses, decimals and %. The percent sign is interpreted as a percentage (e.g., 50% becomes 50/100 when evaluated). Multiplication is shown as × but evaluated as * internally. Press = to compute the result — the output appears in the small result box next to the equals button. Use the C button to clear and ⌫ to delete the last character.

Sets (A / B): The sets area lets you build two letter-sets (A and B) — select the active set then click letters A–Z to add or remove them. Use the operation buttons to compute Union (A ∪ B), Intersection (A ∩ B), or Difference (A - B). Results display in the set result box. Use Clear A or Clear B to empty a set at any time.
